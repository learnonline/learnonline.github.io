LEARNING ANALYTICS

Welcome to this OER (Open Educational Resource) on learning analytics!

This OER will explore Learning Analytics in Adult, Higher, and Corporate Education. Learning analytics provides a multitude of opportunities with which we can collect, measure, and analyze data in order to better understand our students and to improve learning.

This OER is intended to be used by anybody interesting to learn more on learning analytics. It is meant as an introduction to this fascinating topics.

This OER can also be used as part of a collaborative learning experience. For this, we have added a “food for thought” at the end of each topic that can be used for further discussion in a learning group, but also for in-depth self study.
